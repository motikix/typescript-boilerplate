typescript-boilerplate
==
Simple typescript boilerplate.

features
--
* Linter:: [eslint](https://eslint.org/)
* Fixer:: [prettier](https://prettier.io/)
* Test:: [jest](https://jestjs.io/ja/)
* Bundler:: [webpack](https://webpack.js.org/)

how to use
--

### `build`
    yarn run build

### `test`
    yarn run test
